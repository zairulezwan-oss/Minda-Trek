# 🧠 Minda Trek — Program Latihan di Rumah (TCK HTPN)

Aplikasi web interaktif untuk program latihan Terapi Cara Kerja (TCK) kanak-kanak di rumah.

## 🌐 Akses Aplikasi

👉 **[Klik untuk buka Minda Trek](https://YOUR-USERNAME.github.io/mindatrek)**

> Gantikan `YOUR-USERNAME` dengan nama pengguna GitHub anda.

---

## ✨ Ciri-ciri

- 🎮 **11 Aktiviti Terapi** — Ingatan, Jejak & Tulis, Kira, Padanan, Susun, Fokus, ADL, Sosial, Tingkah Laku, Konsep Asas, Penjejakan Visual
- 📊 **Laporan Kemajuan** — Harian, Mingguan, Bulanan + Perbandingan Sebelum/Selepas
- 🇲🇾🇬🇧 **Dwibahasa** — Bahasa Melayu & Inggeris
- ⏱️ **Had Masa Skrin** — 2 jam/hari dengan amaran
- 📱 **Mesra Mudah Alih** — Berfungsi pada telefon & tablet

## 🏥 Untuk Siapa?

- Kanak-kanak berumur 5–12 tahun dengan keperluan khas
- Program home program di bawah pengawasan Terapi Cara Kerja (TCK) HTPN
- Ibu bapa / penjaga sebagai fasilitator

## 📋 Nota Klinikal

Laporan yang dijana adalah untuk **pemantauan home program sahaja** dan bukan laporan klinikal rasmi. Sila berbincang dengan TCK anda untuk penilaian profesional.
